# Mooder

> *Status: Inception — Vision & backlog in progress*

Mooder is a multi-platform solution composed of:

| Component | Technology | Description |
|---|---|---|
| **backend** | Java Quarkus · PostgreSQL | Core API & data backbone |
| **frontend-desktop** | Next.js | Desktop/web client application |
| **ios-app** | Swift (iOS) | Native iOS application |

---

## Repository layout

```
mooder/
├── src/
│   ├── backend/              Java Quarkus backend service
│   ├── frontend-desktop/     Desktop client
│   └── ios-app/              iOS Swift application
├── backlog/              Product backlog, epics & sprints
├── mock-data/            Local seed data
└── docs/                 Project & collaboration documentation
```
